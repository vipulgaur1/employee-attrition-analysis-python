# Employee Attrition Analysis & Business Insights

## 📌 Problem Statement
Employee attrition leads to increased hiring costs and productivity loss.  
This project analyzes employee data to identify key factors driving attrition and suggest actionable retention strategies.

## 📊 Dataset
IBM HR Analytics dataset (~1,400 employees)

## 🛠 Tools & Technologies
- Python (Pandas, NumPy)
- Matplotlib, Seaborn

## 🔍 Key Analysis
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Attrition segmentation by department, income, overtime, and satisfaction
- Correlation and pattern analysis

## 🚨 Key Insights
- Overtime employees show ~3x higher attrition  
- Poor work-life balance leads to ~30% attrition  
- Sales and HR departments have highest attrition rates  
- Low-income roles show significantly higher churn  

## 💡 Business Recommendations
- Reduce overtime workload through better planning  
- Improve work-life balance policies  
- Provide incentives for high-risk roles  
- Focus retention strategies on low-income employees
