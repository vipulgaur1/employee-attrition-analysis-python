# Employee Attrition Analysis & Insights

## Problem Statement
Analyze employee attrition and identify key factors driving employee turnover.

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn

## Key Insights
- Overtime employees show ~3x higher attrition
- Poor work-life balance leads to ~30% attrition
- Sales and HR roles have highest attrition
- Low-income employees show higher churn

## Conclusion
This project provides data-driven insights to improve employee retention strategies.
